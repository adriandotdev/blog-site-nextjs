# The Daily Bytes

<p>Daily Bytes is a blog platform that supports markdown blocks similar to Notion. I integrated Tiptap Editor and apply what I have learned at work using NextJS.</p>

## Tech Stack
- NextJS
- Drizzle
- PostgreSQL
- TailwindCSS
- Supabase
