# The Daily Bytes

The Daily Bytes is a blog platform integrated with the Tiptap editor that supports Markdown blocks similar to those in Notion.

## Tech Stack
- NextJS
- Drizzle
- PostgreSQL
- TailwindCSS
- Supabase
- Google OAuth
